# Analyse du Churn et Performance RH

## Contexte
Analyse de l’attrition des employés sur un dataset RH (IBM).

## Objectifs
- Identifier les profils à risque de départ
- Comprendre les facteurs de churn
- Aider à la prise de décision RH

## Outils
- Power BI
- DAX
- Power Query

## Analyses
- Nettoyage et modélisation des données RH
- Calcul du taux d’attrition
- Analyse salaire / satisfaction / ancienneté

## Résultats
- Identification des segments à fort risque
- Corrélation entre satisfaction et churn
- Insights actionnables pour la rétention
